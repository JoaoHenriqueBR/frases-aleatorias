# Frases Aleatórias de Extrema Sabedoria

Projeto de um Gerador de Frases Aleatórias feito para o Ensino de React e Consumo de APIs, aplicado no Ctrl+Young 3.

## Como executar

### Instalar dependências

- `npm install`

### Iniciar o projeto 

- `npm start`

Abre [http://localhost:3000](http://localhost:3000) no seu navegador
